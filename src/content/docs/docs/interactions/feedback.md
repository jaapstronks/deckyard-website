# Feedback

Gather audience feedback to improve your presentations.

## Overview

The feedback feature helps you understand how your presentation was received and what you can improve. Collect ratings, comments, and reactions from your audience.

![Lead capture slide editor showing form fields for name, email, submit button, thank you message, and privacy consent](/images/screenshots/lead-capture-editor-full.png)

## Feedback Types

### Session Ratings

Overall presentation rating:
- Star ratings (1-5)
- Simple thumbs up/down
- Net Promoter Score (NPS)
- Custom scales

### Slide Feedback

Per-slide reactions and comments:
- Rate individual slides
- Flag confusing content
- Highlight great slides
- Suggest improvements

### Exit Surveys

End-of-presentation questionnaires:
- Multiple questions
- Mix of rating and text
- Optional for audience
- Higher completion with incentive

### Real-time Reactions

Emoji and sentiment during presentation:
- Quick emoji reactions
- Sentiment indicators
- "Slow down" / "Speed up" signals
- Engagement pulse

## Creating a Feedback Slide

![Feedback slide editor with question and QR code preview](/images/screenshots/feedback-slide-editor.png)

### Add Feedback Slide

1. Click **Add Slide**
2. Select **Feedback** from the Interaction category
3. Configure your feedback question
4. Customize the placeholder text
5. Choose a background color

### Feedback Styles

**Star Rating:**
- 1-5 stars
- Clear and familiar
- Good for overall ratings

**Scale:**
- Numeric scale (1-10)
- Good for detailed measurement
- Can label endpoints

**Emoji:**
- Visual emotion selection
- Quick and engaging
- Good for sentiment

**Text Comment:**
- Open-ended feedback
- Rich insights
- More effort for audience

### Multi-Question Feedback

Create surveys with multiple questions:
1. Add Feedback slide
2. Click **Add Question**
3. Mix rating and text questions
4. Maximum 5 questions recommended

## Collecting Feedback

### During Presentation

Show feedback slide and:
- Audience responds via follow mode
- Results update in real-time
- Control when to show results

### After Presentation

Redirect viewers to feedback:
- Auto-show feedback at end
- Send link via email
- Embed in follow-up page

### Reminder Options

Encourage responses:
- Verbal reminder to complete
- Show completion percentage
- Thank responders

## Viewing Results

### Real-time View

During presentation:
- Aggregate scores update live
- Word clouds for text responses
- Sentiment indicators

### Analytics Dashboard

After presentation:
1. Go to presentation analytics
2. Select **Feedback** tab
3. View all responses

### Feedback Report

Report includes:
- Average ratings per question
- Distribution charts
- Text response summary
- Comparison to previous sessions

## Response Analysis

### Quantitative

For ratings and scales:
- Average score
- Standard deviation
- Distribution histogram
- Trend over time

### Qualitative

For text responses:
- All comments listed
- Word cloud visualization
- Sentiment analysis (if AI enabled)
- Theme grouping

### Comparative

Compare across presentations:
- Score trends
- Best/worst performing slides
- Audience segment differences

## Privacy and Anonymity

### Anonymous Feedback

By default, feedback is anonymous:
- No user identification
- Honest responses encouraged
- Privacy respected

### Identified Feedback

If needed:
- Enable identification
- Useful for follow-up
- May reduce response rate
- Clear disclosure required

### Data Retention

Configure retention:
- Keep indefinitely
- Auto-delete after X days
- Anonymize after analysis

## Best Practices

### Question Design

- Keep questions short
- One concept per question
- Use consistent scales
- Test with colleagues

### Timing

- Don't interrupt flow for feedback
- End of presentation is ideal
- Keep it brief (under 2 minutes)
- Thank participants

### Acting on Feedback

- Review after each presentation
- Look for patterns across sessions
- Make specific improvements
- Share improvements with future audiences

## Export

### Export Formats

- **CSV** - Spreadsheet analysis
- **JSON** - Integration with tools
- **PDF** - Report format

### What's Included

- All responses
- Timestamps
- Question text
- Anonymized identifiers (if any)
