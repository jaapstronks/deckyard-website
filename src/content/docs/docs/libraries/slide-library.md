# Slide Library

Build a collection of reusable slides for your team. Save your best slides and insert them into any presentation.

![Slide library with personal and team slides](/images/screenshots/slide-library.png)

## Overview

The Slide Library stores complete slides with all their content, styling, and configuration. When you insert a library slide, it creates a copy that you can customize for the new presentation.

## Saving Slides to Library

### From the Editor

1. Open a presentation in the editor
2. Select the slide you want to save
3. Click the **Save to Library** button (bookmark icon) in the slide panel
4. Enter a name for the slide
5. Choose visibility (personal or team)
6. Add optional tags for organization
7. Click **Save**

### What Gets Saved

When you save a slide to the library:
- All text content
- Images (URLs preserved)
- Slide type and layout settings
- Background and styling options
- Speaker notes

### What Doesn't Transfer

- Presentation-specific settings (theme colors come from the target presentation)
- Interactive slide state (poll results, feedback)
- Slide IDs (new IDs are generated on insert)

## Library Scopes

### Personal Library

- **Visibility**: Only you can see and use these slides
- **Use case**: Draft slides, personal templates, work in progress
- **Storage**: Associated with your user account

### Team Library

- **Visibility**: All members of your team can see and use
- **Use case**: Shared templates, approved content, team standards
- **Storage**: Associated with your team/organization

## Browsing and Searching

### Accessing the Library

1. In the editor, click **Insert Slide**
2. Switch to the **Library** tab
3. Browse or search for slides

### Search Features

- **Text search**: Search by slide name and content
- **Tag filter**: Filter by assigned tags
- **Type filter**: Filter by slide type
- **Scope filter**: Show personal, team, or all

### Preview

Click any library slide to see a preview before inserting. The preview shows:
- Slide thumbnail
- Slide name and type
- Tags
- Last updated date
- Original author

## Inserting Slides from Library

### Quick Insert

1. Find the slide in the library browser
2. Click the slide or press Enter
3. The slide is inserted after your current position

### Insert Options

When inserting, you can choose:
- **After current slide** (default)
- **Before current slide**
- **At end of presentation**

### After Insertion

The inserted slide is a copy. You can:
- Edit any content freely
- Change the slide type
- Delete without affecting the library version

## Managing Library Slides

### Editing Library Slides

To update a library slide:
1. Browse to the slide in the library
2. Click the **Edit** button
3. Modify name, tags, or visibility
4. Click **Save**

Note: You cannot edit the slide content directly in the library. To update content, save a new version from a presentation.

### Favorites

Mark frequently used slides as favorites:
1. Hover over any library slide
2. Click the star icon
3. Favorites appear at the top of the library

### Deleting Library Slides

1. Browse to the slide in the library
2. Click the **Delete** button
3. Confirm deletion

Note: Deleting a library slide does not affect slides already inserted into presentations.

## Theme Filtering

Library slides are designed with specific themes. When browsing:

- **Matching theme**: Slides that match your current presentation's theme appear first
- **Other themes**: Slides from different themes are shown but may need style adjustments

### Theme Compatibility

When inserting a slide from a different theme:
- Text content transfers fully
- Background colors may differ
- Images are preserved
- Layout structure is maintained
- Theme-specific styling adapts to the target theme

## Best Practices

### When to Save

Save slides when:
- You've created a well-designed slide worth reusing
- You want to standardize a particular format
- Team members frequently recreate similar slides
- You have approved brand content

### Naming Conventions

Use clear, descriptive names:
- ✅ "Q4 Sales Summary Template"
- ✅ "Team Introduction - 4 Person"
- ❌ "Slide 1"
- ❌ "test"

### Tagging Strategy

Develop consistent tags:
- By category: `sales`, `marketing`, `hr`
- By use: `template`, `final`, `approved`
- By type: `title`, `content`, `data`

### Library Maintenance

Regularly review your library:
- Remove outdated slides
- Update slides with new branding
- Consolidate duplicates
- Verify team slides are still relevant
