# Editing Presentations

The Deckyard editor is where you build and refine your presentations.

## Editor Layout

The editor has three columns:

1. **Slides panel** (left) - Your slide list for navigation and reordering
2. **Edit panel** (center) - Fields for the selected slide
3. **Preview panel** (right) - Live preview of the current slide

## Presentation Settings

Click the **Settings** button (⚙) in the toolbar to configure:

- **Enable Q&A** - Allow audience questions
- **Builds** - Reveal content step-by-step while presenting
- **Presenter transition** - Slide transition effects (Fade, Slide, Push, 3D Cube)
- **Document language** - Language for exports and sharing
- **Tags** - Organize and filter your presentations
- **Description** - Meta description for published presentations

![Deck settings dialog with theme, transitions, and engagement options](/images/screenshots/deck-settings.png)

## Auto-Save

Changes are saved automatically as you work. If you try to leave with unsaved changes, you'll see a browser warning.

## Language Switching

The editor works in one language at a time. Use the language toggle at the top to switch between NL and EN. Each language version is edited separately.
