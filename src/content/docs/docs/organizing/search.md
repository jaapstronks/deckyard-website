# Search

Quickly find presentations using Deckyard's search features.

## Full-Text Search

![Search results showing matching presentations](/images/screenshots/search-results.png)

Search across:

- Presentation titles
- Presentation descriptions
- Slide content

Type in the search box to filter the presentation list in real-time.

## Deep Slide Search

Find presentations containing specific content within slides:

- Search finds text inside any slide
- Results show which presentations contain your search term
- Useful for finding "that presentation with the budget slide"

## Searching by Tag

Click a tag to filter presentations:

- Shows only presentations with that tag
- Combine with search for precise filtering

See [Tags](/docs/organizing/tags) for more on tagging.

## Presentation Scopes

Search respects visibility:

- **Private** - Only your presentations
- **Workspace** - Shared team presentations
- **Published** - Publicly accessible presentations

## Tips

- Use specific terms for better results
- Search works in both NL and EN content
- Recent presentations appear at the top by default
