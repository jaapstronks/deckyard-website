# Notion Integration

Connect Deckyard to Notion to import content from your workspace.

## Setup

Add your Notion integration secret to `.env`:

```
NOTION_SECRET=secret_xxxxxxxxxxxxxxxxxxxxxxxxx
```

You can create an integration at [notion.so/my-integrations](https://www.notion.so/my-integrations).

## Usage

Once configured, see [Creating from Notion](/docs/creating/from-notion) for how to import content from Notion.
