# Export Formats

Export your presentations for different purposes.

## Overview

Deckyard supports exporting presentations to multiple formats for offline use, sharing, and integration with other tools.

## Available Formats

![Export menu with all available formats](/images/screenshots/export-menu.png)

### PDF

Standard document format for sharing and printing.
- High-quality rendering
- Universal compatibility
- Print-ready output

[Learn more about PDF export](/docs/export/pdf/)

### PowerPoint (PPTX)

Microsoft PowerPoint format for editing in other tools.
- Editable slides
- Compatible with PowerPoint, Google Slides, Keynote
- Preserves basic formatting

[Learn more about PowerPoint export](/docs/export/pptx/)

### Images (PNG)

Export slides as individual images.
- High-resolution output
- Perfect for social media
- Great for thumbnails

[Learn more about image export](/docs/export/images/)

### HTML

Self-contained web presentations.
- Offline viewing
- No internet required
- Full interactivity preserved

### Notes

Export speaker notes separately.
- Word document (DOCX)
- Markdown format
- For printing or reference

[Learn more about notes export](/docs/export/notes/)

### Handoff Bundle

Design handoff package with assets.
- All images included
- Style information
- For designers and developers

[Learn more about handoff bundles](/docs/export/handoff-bundle/)

## Getting Started

### Quick Export

1. Open your presentation
2. Click **File** > **Export**
3. Select your format
4. Configure options
5. Click **Export**

### Export Menu

Access all export options from:
- **File** menu > **Export**
- Keyboard shortcut: `Ctrl/Cmd + E`
- Right-click menu on presentation card

## Export Permissions

### Who Can Export

- **Owner** - All export formats
- **Admin** - All export formats
- **Editor** - Most export formats
- **Viewer** - Depends on presentation settings

### Restricting Export

Owners can restrict export:
1. Go to presentation **Settings**
2. Find **Export Options**
3. Disable specific formats or all exports

## Common Options

These options are available across export formats:

### Language Selection

For multi-language presentations:
- Select which language to export
- Or export all languages

### Slide Selection

Export specific slides:
- All slides
- Current slide
- Custom range

### Quality Settings

Adjust output quality:
- **Standard** - Good balance of quality and size
- **High** - Maximum quality
- **Web** - Optimized for web sharing
