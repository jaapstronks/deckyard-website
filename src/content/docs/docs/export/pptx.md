# PowerPoint Export

Export your presentations as PowerPoint files for use in Microsoft PowerPoint or other compatible applications.

## How to Export

### From the Editor

1. Open your presentation
2. Click **File** > **Export** > **PowerPoint**
3. Configure export options
4. Click **Export**
5. Download the PPTX file

### From the Presentation List

1. Right-click on a presentation
2. Select **Export** > **PowerPoint**
3. Configure options and export

## Export Options

### Image Quality

Choose image resolution:
- **Standard** - Good balance of quality and file size
- **High** - Better quality, larger file

### Include Notes

- **Yes** - Speaker notes included in PowerPoint notes
- **No** - Slides only

### Language

For multi-language presentations:
- Select which language version to export
- Content exports in the selected language

## What's Exported

### Fully Supported

- Text content and formatting
- Images and photos
- Basic shapes
- Speaker notes
- Slide order

### Partially Supported

- Charts (converted to images)
- Complex layouts (simplified)
- Custom fonts (may substitute)

### Not Supported

- Animations and transitions
- Videos (poster frame only)
- Interactive elements
- Custom slide types (converted to images)

## Limitations

### Layout Differences

PowerPoint uses different layout models. Some layouts may appear differently:
- Text boxes may be positioned differently
- Margins and padding may vary
- Complex designs convert to images

### Font Substitution

If fonts aren't available in PowerPoint:
- System fonts are substituted
- Text layout may shift
- Consider using common fonts

### Slide Type Conversion

Complex Deckyard slide types are converted:
- Charts become images
- Interactive slides become static
- Custom components become images

### Theme Limitations

Themes are partially converted:
- Colors are preserved
- Fonts may substitute
- Background images work
- Complex gradients may simplify

## Compatibility

### Tested With

- Microsoft PowerPoint 2016 and later
- Microsoft 365
- Google Slides
- Apple Keynote
- LibreOffice Impress

### Best Compatibility

For best results:
- Use simple layouts
- Stick to common fonts
- Avoid complex custom slides

## Use Cases

### Editing Elsewhere

When you need to:
- Edit in PowerPoint
- Share with PowerPoint users
- Use corporate templates

### Offline Presenting

When presenting without Deckyard:
- Conferences with limited internet
- Corporate environments with restrictions
- Backup presentation format

### Handoff

When handing off to others:
- Clients who use PowerPoint
- Teams with different tools
- Archive in common format

## Tips for Better Export

### Before Exporting

1. Use supported slide types when possible
2. Check fonts are common/web fonts
3. Simplify complex layouts
4. Preview slides before export

### After Exporting

1. Open in PowerPoint and review
2. Check text alignment
3. Verify images loaded
4. Test any animations you add in PowerPoint

## Troubleshooting

### Slides Look Different

This is expected due to format differences:
- Recreate complex layouts manually
- Adjust positioning in PowerPoint
- Accept some simplification

### Missing Fonts

Install the required fonts or:
- Use font substitution
- Change fonts in Deckyard before export
- Edit in PowerPoint after export

### Large File Size

To reduce size:
- Choose Standard image quality
- Optimize images in Deckyard first
- Remove unused slides

### Elements Missing

Interactive or custom elements don't export:
- They appear as screenshots
- Add functionality in PowerPoint if needed
