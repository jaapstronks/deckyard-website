# Create from PowerPoint

Import existing PowerPoint (.pptx) presentations into Deckyard.

## How It Works

1. From the presentation list, select **Import**
2. Choose **PowerPoint (.pptx)**
3. Upload your file
4. Select a theme to apply
5. Click **Import**

## What Gets Converted

- **Text content** - Titles, body text, and bullet points
- **Basic layouts** - Mapped to equivalent Deckyard slide types
- **Images** - Imported and placed on slides
- **Speaker notes** - Preserved per slide

## Theme Application

During import, choose a Deckyard theme. The theme's styles (colors, fonts, layouts) replace the original PowerPoint formatting, ensuring consistency with your brand.

## Limitations

Some PowerPoint features don't have direct equivalents:

- Complex animations are simplified
- Custom shapes may not convert perfectly
- Embedded videos need to be re-added
- SmartArt is converted to basic layouts

## After Import

1. Review each slide for conversion accuracy
2. Adjust layouts where needed
3. Re-add any media that didn't convert
4. Generate a translation if needed
