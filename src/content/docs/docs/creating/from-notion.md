# Create from Notion

Turn your Notion pages into presentations with a few clicks.

## How It Works

![New presentation dialog with From content option selected, showing Paste text and Upload file tabs with theme and language selection](/images/screenshots/new-presentation-from-content.png)

1. Open the **AI Wizard** from the presentation list
2. Select **"From Notion Page"**
3. Search for and select the Notion page you want to import
4. Choose output language (NL or EN)
5. Click **Generate**

## What Gets Imported

The system extracts and converts:

- **Headings** - Become slide titles and section breaks
- **Text paragraphs** - Become content slides
- **Bullet lists** - Become list slides
- **Numbered lists** - Preserved as ordered content
- **Quotes** - Become quote slides
- **Code blocks** - Preserved with formatting
- **Callouts** - Converted to appropriate slide types
- **Tables** - Become table slides
- **Images** - Imported as image slides

## After Import

Review and refine:

1. Check that content was split into slides logically
2. Adjust slide types if the AI chose incorrectly
3. Add images where needed
4. Write speaker notes
5. Generate the translation if presenting bilingually

## Publishing Back to Notion

If your presentation was created from a Notion page, you can embed it back:

1. Go to export options
2. Select **"Add to Notion Page"**
3. The embed is automatically added to the source page

## Notion Integration Setup

Your Deckyard instance needs the Notion integration configured. See [Notion Integration](/docs/integrations/notion) for setup instructions.
