Export your speaker notes as standalone documents for printing or reference.

## Export Formats

### DOCX (Microsoft Word)

- Professional document format
- Editable in Word, Google Docs, etc.
- Includes formatting and structure

### Markdown

- Plain text Markdown format
- Version control friendly
- Universal compatibility

## How to Export

### From the Editor

![Export dropdown menu showing PDF, PNG, PPTX, HTML, JSON, Print text, Notes, and Handoff ZIP options](/images/screenshots/export-dropdown-full.png)

1. Open your presentation
2. Click **Export** > **Notes**
3. Select format (DOCX or Markdown)
4. Configure options
5. Click **Export**

### From the Export Menu

1. Click **File** > **Export**
2. Choose **Speaker Notes**
3. Select your preferred format

## Export Options

### Include Slides

- **Notes only** - Just the speaker notes text
- **With thumbnails** - Include small slide images
- **With slide text** - Include slide content for context

### Structure

- **By slide** - Notes organized by slide number
- **Continuous** - All notes as flowing text
- **With headings** - Slide titles as section headers

### Language

For multi-language presentations:
- Select which language to export
- Notes export in the selected language

## Document Structure

### DOCX Output

The Word document includes:
- Document title (presentation name)
- Slide number and title for each section
- Speaker notes content
- Optional slide thumbnails
- Page numbers

### Markdown Output

```markdown
# Presentation Title

## Slide 1: Introduction

Speaker notes for the first slide go here...

## Slide 2: Main Topic

Notes for the second slide...
```

## Use Cases

### Printing

Print notes for:
- Physical reference during presentation
- Handouts with additional context
- Study materials

### Sharing

Share notes separately:
- Send context to team members
- Provide speaker guidance
- Document talking points

### Backup

Keep notes accessible:
- Archive with other documents
- Include in version control
- Back up presentation content

### Script Development

Use exported notes to:
- Develop full presentation script
- Create video narration
- Write accompanying articles

## Tips for Better Notes

### Writing Notes

- Keep notes concise
- Use bullet points for key points
- Include timing cues
- Add transition reminders

### Formatting

Notes support basic formatting:
- **Bold** for emphasis
- *Italic* for secondary points
- Bullet lists
- Numbered lists

### Organization

- One key point per paragraph
- Use blank lines for pauses
- Mark important transitions
