Export your presentations as PDF files for offline viewing and printing.

## How to Export

### From the Editor

![Export dropdown menu showing PDF, PNG, PPTX, HTML, JSON, Print text, Notes, and Handoff ZIP options](/images/screenshots/export-dropdown-full.png)

1. Open your presentation
2. Click **Export** > **PDF**
3. Configure export options
4. Click **Export**
5. Download the PDF file

### From the Presentation List

1. Right-click on a presentation
2. Select **Export** > **PDF**
3. Configure options and export

### Keyboard Shortcut

Press `Ctrl/Cmd + Shift + P` to open PDF export directly.

## Export Options

### Page Size

Select the output size:
- **16:9** - Widescreen (default)
- **4:3** - Standard
- **A4** - Standard paper size
- **Letter** - US paper size
- **Custom** - Enter custom dimensions

### Quality

Choose output quality:
- **Standard** (72 DPI) - Smaller file, good for screen viewing
- **High** (150 DPI) - Good for printing
- **Print** (300 DPI) - Professional print quality

### Slides per Page

Layout options:
- **1 per page** - Full slide view (default)
- **2 per page** - Two slides stacked
- **4 per page** - Grid layout
- **6 per page** - Handout style

### Include Options

What to include:
- **Slide numbers** - Show page numbers
- **Speaker notes** - Add notes below slides
- **Outline** - Include slide titles as bookmarks

### Language

For multi-language presentations:
- Select which language version to export
- Each language creates a separate PDF

## Advanced Options

### Print-Optimized Mode

Enable for professional printing:
- CMYK color conversion (where supported)
- Crop marks
- Bleed area

### Accessibility

Improve PDF accessibility:
- **Tagged PDF** - Structure for screen readers
- **Alt text** - Include image descriptions
- **Reading order** - Logical content order

## What's Included

### Preserved

- All text and formatting
- Images and graphics
- Charts and diagrams
- Theme colors and fonts
- Background images

### Not Included

- Animations (shown in final state)
- Videos (shown as poster image)
- Interactive elements (polls, Q&A)
- Embedded web content

## Use Cases

### Sharing

- Send to colleagues without Deckyard access
- Email attachments
- Archive presentations

### Printing

- Handouts for meetings
- Physical copies for events
- Backup documentation

### Review

- Offline review and markup
- Legal or compliance archives
- Version snapshots

## File Size

PDF size depends on:
- Number of slides
- Image count and resolution
- Quality setting chosen

Tips for smaller files:
- Use Standard quality for screen viewing
- Optimize images before adding to presentation
- Avoid very high-resolution backgrounds

## Troubleshooting

### Export Takes Too Long

For large presentations:
- Try Standard quality first
- Export in smaller batches
- Check your internet connection

### Fonts Look Different

PDF may substitute fonts if:
- Custom fonts aren't embeddable
- Fonts aren't available on the server

Solution: Use web-safe fonts or system fonts.

### Images Are Low Quality

Increase export quality:
1. Select **High** or **Print** quality
2. Ensure source images are high resolution
3. Re-export
