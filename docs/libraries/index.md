# Asset Libraries

Save and reuse slides and images across your presentations. Libraries help maintain consistency and speed up presentation creation.

## Overview

Deckyard's asset libraries provide a central repository for reusable content that can be shared across presentations and teams.

![Slide library showing saved slides with Personal/Team tabs, language filter, search, and Use button](/images/screenshots/slide-library-overview.png)

### Available Libraries

| Library | Purpose |
|---------|---------|
| [Slide Library](/docs/libraries/slide-library/) | Save and reuse complete slides |
| [Image Library](/docs/libraries/image-library/) | Manage uploaded images |

## Benefits

### Consistency
- Maintain brand consistency across all presentations
- Ensure standardized formatting and styling
- Use approved content and imagery

### Efficiency
- Reuse proven slides instead of rebuilding from scratch
- Quick access to frequently used images
- Search and filter to find assets fast

### Collaboration
- Share assets with your team
- Build a library of approved content
- Reduce duplication of effort

## Library Scopes

Assets can be saved at different visibility levels:

| Scope | Visibility | Best For |
|-------|------------|----------|
| Personal | Only you | Draft slides, personal photos |
| Team | Your team members | Shared templates, team photos |
| Organization | All users | Brand assets, approved content |

## Getting Started

### Building Your Slide Library

1. Create a well-designed slide in any presentation
2. Click the "Save to Library" button in the slide panel
3. Choose visibility scope (personal/team)
4. Add tags for easy discovery
5. Insert the slide into any future presentation

### Building Your Image Library

1. Upload images through the editor
2. Images are automatically added to your library
3. Access from any presentation's image picker
4. Search by filename or browse recent uploads

## Best Practices

### Slide Library

- **Quality over quantity** - Only save polished, reusable slides
- **Use descriptive names** - Make slides easy to find
- **Tag consistently** - Develop a tagging taxonomy
- **Review periodically** - Remove outdated content

### Image Library

- **Optimize images** - Compress before uploading
- **Use meaningful filenames** - Helps with search
- **Remove unused images** - Keep library manageable
- **Respect copyrights** - Only upload licensed content
