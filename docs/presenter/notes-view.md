Access your speaker notes on a secondary device while presenting.

## Overview

The notes view lets you see your speaker notes on a phone or tablet while presenting on the main screen. It's perfect for walking around during presentations or presenting without a laptop in front of you.

## Opening Notes View

### From Presenter Mode

1. Start presenter mode
2. Click **Notes View** or **Mobile Remote**
3. Scan the QR code with your phone
4. Notes view opens on your device

### Direct URL

1. Open presentation settings
2. Find the **Notes View URL**
3. Open this URL on any device
4. You're connected

### QR Code

Display a QR code for easy connection:
1. Click **Companion** in the editor toolbar
2. QR code appears on screen
3. Scan with your phone's camera
4. Notes view opens automatically

## Notes View Interface

### Current Slide Notes

- Large, readable text
- Current slide's notes displayed
- Auto-scrolls with slide changes

### Navigation Controls

Control your presentation from the notes view.

**Next/Previous:**
- Swipe left/right
- Tap navigation buttons
- Use hardware buttons (if configured)

**Jump to Slide:**
- Tap slide counter
- Select from slide list

### Current Slide Preview

- Small preview of current slide
- Confirms sync is working
- Tap to enlarge briefly

### Timer

- Elapsed time display
- Matches presenter mode timer
- Visible at all times

## Navigation Controls

### Touch Gestures

| Gesture | Action |
|---------|--------|
| Swipe right | Next slide |
| Swipe left | Previous slide |
| Tap center | Show/hide controls |
| Double tap | Full screen |

### Buttons

- **< Previous** - Go back one slide
- **Next >** - Advance one slide
- **Overview** - See all slides

### Hardware

Some devices support:
- Volume buttons for navigation
- Bluetooth clicker integration
- Physical keyboard shortcuts

## Q&A Monitoring

Monitor incoming questions while presenting.

### Live Q&A Feed

If Q&A is enabled:
- Questions appear in real-time
- New questions highlighted
- Swipe to dismiss

### Moderation

From notes view you can:
- See all submitted questions
- Promote questions to display
- Mark questions as answered
- Dismiss inappropriate questions

### Notification

When new questions arrive:
- Subtle badge indicator
- Optional vibration
- Doesn't interrupt presentation flow

## Settings

### Text Size

Adjust notes text size:
1. Tap settings icon
2. Adjust **Notes Size** slider
3. Find comfortable reading size

### Theme

Choose display theme:
- **Auto** - Matches device setting
- **Light** - White background
- **Dark** - Black background (saves battery)

### Keep Screen On

Enable to prevent screen sleep:
1. Toggle **Keep Screen On**
2. Your device stays awake during presentation

## Syncing

### Connection Status

Indicator shows sync status:
- **Green** - Connected and synced
- **Yellow** - Reconnecting
- **Red** - Disconnected

### Reconnecting

If connection is lost:
1. Notes view attempts auto-reconnect
2. If needed, scan QR code again
3. Slide position is restored

### Latency

Notes view updates in real-time:
- Typically under 100ms delay
- Works on slow connections
- Graceful degradation

## Tips

### Before Presenting

- Test the connection beforehand
- Adjust text size for readability
- Enable "Keep Screen On"
- Check battery level

### During Presenting

- Hold device comfortably
- Glance briefly at notes
- Don't read verbatim
- Maintain audience eye contact

### Technical Setup

- Use reliable WiFi
- Have backup notes (printed)
- Test with actual presentation
