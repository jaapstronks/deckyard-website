Enable your audience to follow your presentation on their own devices.

## Overview

Follow mode provides a companion view for audience members, syncing their devices with your presentation. Perfect for large venues, hybrid events, or when audience members want the content on their personal screens.

## How It Works

When you enable follow mode, audience members can:
- View the current slide on their own device
- Switch languages if translations are available
- Participate in interactive elements (polls, Q&A)
- Access the presentation after you're done

## Sharing the Follow Link

### Getting the Link

1. Start presenting your presentation
2. Click **Share** > **Follow Link**
3. Copy the short URL

### Link Format

Follow links use short codes for easy sharing:
```
https://your-instance.com/f/ABCD
```

Four-letter codes are easy to type and remember.

### Sharing Methods

**Verbally:**
> "Join at [your-domain]/f/ABCD"

**On a slide:**
Add a "Follow Invite" slide with:
- The URL
- QR code
- Instructions

**Via chat/email:**
Share the link in your meeting chat or event materials.

## QR Code Sharing

Display a QR code that audience members can scan to join.

### Displaying QR Code

1. Click **Share** > **Show QR Code**
2. QR code appears on screen
3. Audience scans with phone camera
4. Follow view opens automatically

### On Slides

Use the Follow Invite slide type:
1. Add a new slide
2. Select **Follow Invite** type
3. QR code and link are automatically included
4. Customize the message

![Follow Invite slide with QR code and access code](/images/screenshots/follow-invite-slide.png)

### QR Code Tips

- Make it large enough to scan from the back
- Leave on screen for at least 30 seconds
- Consider showing at the start
- Show again before interactive elements

## Language Switching

For multi-language presentations:

### Audience Language Selection

1. Audience joins via follow link
2. Click the language selector
3. Choose preferred language
4. All content switches to that language

### Available Languages

Only languages with translations appear:
- Primary language always available
- Translated languages shown as options
- Partial translations show what's available

### Real-time Translation

If live translation is running:
- Translated content appears as it's ready
- Audience sees content in their language
- Slides stay in sync with presenter

## Interaction Participation

### Polls

When you show a poll slide:
1. Audience sees voting options
2. They tap to vote
3. Results update in real-time
4. You control when results are revealed

### Q&A

During Q&A:
1. Audience submits questions
2. Questions appear in your moderator view
3. You select questions to address
4. Audience sees featured questions

### Feedback

For feedback slides:
1. Audience provides ratings or comments
2. Submissions are collected
3. Results available in analytics

### Lead Capture

Lead capture forms work through follow mode:
1. Audience fills out the form
2. Data is captured securely
3. GDPR consent is collected

## Viewer Experience

### Mobile-Optimized

Follow view is designed for phones:
- Responsive layout
- Touch-friendly controls
- Fast loading
- Low bandwidth usage

### What Viewers See

- Current slide content
- Navigation (if enabled)
- Interaction controls
- Language selector

### What's Hidden

- Speaker notes
- Future slides (unless enabled)
- Editing controls
- Admin features

## After the Presentation

### Continued Access

Configure what happens after you stop:
- **Allow review** - Viewers can browse slides
- **Lock** - Viewers see "Presentation ended"
- **Redirect** - Send to a thank-you page

### Analytics

Track engagement:
- How many joined
- When they joined/left
- Interaction participation
- Language preferences

## Settings

### Enable/Disable

Control follow mode availability:
1. Go to presentation settings
2. Find **Follow Mode**
3. Toggle on/off

### Code Regeneration

Create new follow codes:
1. Go to **Share** > **Follow Settings**
2. Click **Generate New Code**
3. Old codes stop working
4. Share the new code

### Access Control

Limit who can join:
- **Anyone with link** - Open access
- **Require email** - Collect viewer emails
- **Password protected** - Add a password

## Technical Details

### Rate Limits

Follow codes are protected:
- 10 code creations per hour per IP
- 60 code resolutions per hour per IP
- Prevents abuse

### Bandwidth

Follow mode is efficient:
- Minimal data transfer
- Works on slow connections
- Progressive image loading

### Device Support

Works on:
- All modern smartphones
- Tablets
- Laptops
- Any device with a browser
