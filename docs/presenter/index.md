Take control of your presentations with the presenter view.

## Overview

Presenter mode provides you with tools and information to deliver your presentation effectively. While your audience sees the main presentation, you have access to notes, controls, and timing information.

![Presenter mode with slide view and controls](/images/screenshots/presenter-mode.png)

## Features

- **Speaker Notes** - View your notes while presenting
- **Slide Preview** - See upcoming slides
- **Timer** - Track your presentation time
- **Navigation Controls** - Move through slides easily
- **Laser Pointer & Drawing** - Highlight content on your slides
- **Slide Timing & Pacing** - Auto-advance or pacing timer with per-slide durations
- **Audience View** - See what your audience sees
- **Q&A Monitoring** - Watch for audience questions

## Getting Started

### Starting Presenter Mode

1. Open your presentation
2. Click **Present** in the toolbar
3. Select **Presenter View**
4. A new window opens with presenter controls

### Keyboard Shortcut

Press `P` while viewing a presentation to enter presenter mode.

### Two-Window Setup

Presenter mode works best with two screens:
- **Main screen** - Audience view (full screen)
- **Secondary screen** - Presenter controls

Move the presenter window to your laptop and project the audience view.

## Presenter Interface

### Layout

The presenter view includes:

**Current Slide** (large, left)
- Shows what the audience sees
- Click to advance

**Next Slide** (smaller, right)
- Preview of the upcoming slide
- Helps with smooth transitions

**Speaker Notes** (bottom)
- Your notes for the current slide
- Scrollable for long notes

**Timer** (top)
- Elapsed time
- Optional countdown timer
- Pause/reset controls

**Navigation** (top bar)
- Slide counter
- Previous/next buttons
- Jump to slide

## Navigation

### Advancing Slides

Multiple ways to advance:
- Click on the current slide
- Press `Right Arrow`, `Space`, or `Enter`
- Use the **Next** button
- Click on the next slide preview

### Going Back

- Press `Left Arrow` or `Backspace`
- Use the **Previous** button

### Jump to Slide

- Click the slide counter
- Enter a slide number
- Or select from the slide overview

### Slide Overview

Press `G` to open slide overview:
- See all slides as thumbnails
- Click any slide to jump to it
- Useful for Q&A or skipping sections

## Timer

### Elapsed Timer

Shows how long you've been presenting:
- Starts automatically when you begin
- Pause with the pause button
- Reset to start fresh

### Countdown Timer

Set a target duration:
1. Click on the timer
2. Enter your target time
3. Timer counts down
4. Warning when time is low

### Timing Indicators

- **Green** - Plenty of time
- **Yellow** - Approaching limit
- **Red** - Over time

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Space`, `Right`, `Enter` | Next slide |
| `Left`, `Backspace` | Previous slide |
| `Home` | First slide |
| `End` | Last slide |
| `F` | Toggle fullscreen |
| `L` | Toggle laser pointer |
| `D` | Toggle draw mode |
| `C` | Clear drawings |
| `P` | Toggle persistent drawings |
| `G` | Slide overview |
| `B` | Black screen |
| `A` | Pause/resume timer |
| `N` | Notes toggle |
| `Esc` | Disable highlighter / exit presenter mode |

## Black Screen

Press `B` to show a black screen:
- Hides the presentation
- Useful for breaks
- Keeps timer running
- Press again to resume

## Presenting Without Presenter Mode

If you don't need presenter features:
1. Click **Present** > **Audience View**
2. Full-screen presentation starts
3. Navigate with keyboard

## Notes Companion

Control from your phone:
1. Open your presentation in the editor
2. Click **Companion** in the toolbar
3. Scan QR code with your phone
4. View notes and control slides from your phone

See [Notes View](/docs/presenter/notes-view/) for details.

## Multi-Presenter

For panels or co-presentations:
- Share the presenter link with co-presenters
- Each person can advance slides
- Use locks to coordinate (see [Presentation Locks](/docs/presenter/presentation-locks/))
