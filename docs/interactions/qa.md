Collect and manage audience questions during your presentation.

## Overview

The Q&A feature allows audience members to submit questions that you can address during or after your presentation. With moderation and upvoting, you can focus on the most relevant questions.

![Interaction slide types in the slide picker: Poll, Likert, Likert slider, Feedback, and Follow-along invite](/images/screenshots/slide-types-interaction.png)

## Features

- **Question Submission** - Audience can submit questions via their devices
- **Moderation** - Review and approve questions before displaying
- **Upvoting** - Audience can vote on questions they want answered
- **Live Display** - Show selected questions on screen

## Setting Up Q&A

### Enable Q&A

1. Go to presentation **Settings**
2. Find **Interactions** section
3. Toggle **Q&A** on

### Q&A Settings

**Moderation:**
- **Pre-moderation** - Review before visible to audience
- **Post-moderation** - Visible immediately, can remove
- **No moderation** - All questions visible

**Anonymous Questions:**
- Allow anonymous submissions
- Or require name/email

**Upvoting:**
- Enable audience upvoting
- Questions with more votes rise to top

## Audience Submission

### Via Follow Mode

1. Audience joins presentation via follow link
2. Finds Q&A panel or slide
3. Types their question
4. Submits (optionally with name)

### Submission Form

Audience sees:
- Text field for question
- Optional name field
- Submit button
- Confirmation after submission

## Moderation

### Moderator Panel

As presenter, you see:
- All submitted questions
- Upvote counts
- Submission time
- Name (if provided)

### Actions

For each question:
- **Promote** - Display on screen
- **Answer** - Mark as answered
- **Dismiss** - Remove from queue
- **Hide** - Keep but don't display

### Moderation Queue

Questions are sorted by:
- Upvote count (highest first)
- Time submitted (newest or oldest)
- Status (pending, promoted, answered)

### Mobile Moderation

Moderate from your phone via Notes View:
1. Open notes view on your device
2. Q&A panel shows incoming questions
3. Swipe to promote or dismiss
4. Works alongside speaker notes

## Upvoting

Audience can upvote questions they want answered.

### How It Works

1. Audience sees list of questions
2. Tap upvote button on questions they like
3. Can only upvote each question once
4. Most upvoted rise to top

### Benefits

- Surfaces popular questions
- Democratic prioritization
- Reduces duplicate questions
- Engages passive audience

## Live Display

### Displaying Questions

When you promote a question:
- It appears on the presentation screen
- Audience on follow mode sees it
- You can read and address it

### Display Options

**Overlay:**
- Question appears as overlay
- Your content stays visible
- Good for quick questions

**Full Slide:**
- Question takes full screen
- Good for focused discussion
- Navigate away when done

**Dedicated Slide:**
- Add a Q&A slide type
- Shows promoted question
- Queue of upcoming questions

## During Presentation

### Workflow

1. Audience submits questions throughout
2. You check moderator panel periodically
3. Promote interesting questions
4. Address them verbally or on screen
5. Mark as answered

### Timing Options

**Continuous:**
- Accept questions throughout
- Address when convenient
- Good for informal sessions

**Q&A Segment:**
- Save questions for dedicated time
- Show Q&A slide
- Address accumulated questions

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Q` | Open Q&A moderator panel |
| `N` | Next question |
| `A` | Mark current as answered |
| `D` | Dismiss current question |

## After Presentation

### Export Questions

Export all questions:
- CSV format
- Includes question text, name, votes
- Answered/unanswered status
- Timestamps

### Follow-Up

Use exports to:
- Answer unanswered questions via email
- Create FAQ document
- Plan future content
- Identify knowledge gaps

## Best Practices

### Before the Session

- Test Q&A with colleagues
- Prepare for common questions
- Have a moderator if expecting many questions
- Set clear expectations with audience

### During the Session

- Acknowledge questions are coming in
- Thank people for good questions
- Group similar questions
- Manage expectations on timing

### Question Types

Handle different question types:
- **Scope questions** - Save for later
- **Off-topic** - Acknowledge, defer
- **Complex** - Offer to follow up
- **Hostile** - Address professionally or defer
