Navigate Deckyard efficiently with keyboard shortcuts.

## Editor Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + S` | Save (auto-saves, but confirms) |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |
| `Cmd/Ctrl + F` | Search slides |
| `Arrow Up/Down` | Navigate slide list |

## Lightbox Shortcuts

| Shortcut | Action |
|----------|--------|
| `←` / `→` | Previous / next slide |
| `Escape` | Close popup, or close lightbox |

Arrow keys are disabled when a comment text field is focused.

## Presenter Shortcuts

| Shortcut | Action |
|----------|--------|
| `→` or `Space` | Next slide/step |
| `←` | Previous slide/step |
| `Home` | First slide |
| `End` | Last slide |
| `F` | Toggle fullscreen |
| `Escape` | Exit fullscreen or return to editor |

## Text Editing

Standard text editing shortcuts work in all fields:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + B` | Bold (in rich text fields) |
| `Cmd/Ctrl + I` | Italic (in rich text fields) |
| `Cmd/Ctrl + A` | Select all |
| `Cmd/Ctrl + C/V/X` | Copy/paste/cut |

## Touch Navigation

On mobile devices (Notes view):

- **Swipe left/right** - Navigate slides (when remote control enabled)
- **Tap** - Select items
