# Speaker Notes

Speaker notes help you remember what to say during your presentation.

## Adding Notes

Each slide has a **Speaker Notes** field below the main content fields:

1. Select a slide
2. Scroll down to the **Speaker Notes** section
3. Type your notes
4. Notes save automatically

![Editor showing the Presenter notes field at the bottom-right, below the slide preview](/images/screenshots/speaker-notes-in-context.png)

## Markdown Support

Notes support Markdown formatting:

- **Bold** key points with `**text**`
- Create bullet lists with `-`
- Add emphasis with `*italic*`

## Using Notes While Presenting

During a presentation, view your notes on your phone:

1. Click **Notes (QR)** in the editor
2. Scan the QR code with your phone
3. Your phone shows the current slide preview and notes
4. Notes sync automatically as you advance slides

See [Notes View](/docs/presenter/notes-view) for details.

## Best Practices

- **Keep notes concise** - Bullet points work better than paragraphs
- **Include key phrases** - Write what you want to say, not full scripts
- **Add timing cues** - Note transitions like "pause here" or "demo time"
- **Write early** - Add notes while the content is fresh

## Notes in Exports

Speaker notes can be exported separately via [Notes Export](/docs/export/notes) for printing or sharing.
