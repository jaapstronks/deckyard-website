# Activity & Notifications

Keep track of what's happening in your shared presentations.

## Overview

Deckyard tracks activity and sends notifications to keep team members informed about changes, comments, and collaborator actions.

## Activity Feed

View a history of all changes made to a presentation.

![Activity panel showing recent changes to a presentation](/images/screenshots/activity-panel.png)

### Accessing the Activity Feed

1. Open a presentation
2. Click **File** > **Activity** (or the activity icon)
3. The activity panel opens

### What's Tracked

The activity feed shows:
- **Slide changes** - Additions, deletions, reordering
- **Content edits** - Text, images, and media changes
- **Comments** - New comments and resolutions
- **Collaborators** - Added, removed, permission changes
- **Publishing** - Publish and unpublish events
- **Exports** - PDF, PPTX, and other exports

### Activity Details

Each activity entry shows:
- **Who** - User who performed the action
- **What** - Description of the action
- **When** - Timestamp
- **Where** - Which slide (if applicable)

### Filtering Activity

Filter to find specific events:
- By user
- By activity type
- By date range
- By slide

## Notification Bell

Access your notifications from the notification bell in the header.

### Notification Badge

The bell icon shows:
- **Red badge** - Number of unread notifications
- **No badge** - All caught up

### Notification Panel

Click the bell to open the panel:
- List of recent notifications
- Mark as read/unread
- Click to navigate to the related item
- Clear all notifications

### Notification Types

You receive notifications for:
- **Mentions** - Someone mentioned you in a comment
- **Comments** - New comments on your presentations
- **Sharing** - Someone shared a presentation with you
- **Ownership** - Presentation ownership transferred to you
- **Publishing** - Presentations you collaborate on were published

## Event Types

Events that are tracked include:

### Content Events

| Event | Description |
|-------|-------------|
| Slide added | New slide created |
| Slide deleted | Slide removed |
| Slide reordered | Slides moved in order |
| Content edited | Text or media changed |
| Theme changed | Presentation theme updated |

### Collaboration Events

| Event | Description |
|-------|-------------|
| Collaborator added | New person invited |
| Collaborator removed | Access revoked |
| Permission changed | Access level modified |
| Comment added | New comment posted |
| Comment resolved | Comment marked resolved |
| Mention | You were @mentioned |

### Publishing Events

| Event | Description |
|-------|-------------|
| Published | Presentation made public |
| Unpublished | Public access removed |
| Share link created | New share link generated |

### Export Events

| Event | Description |
|-------|-------------|
| PDF exported | PDF download |
| PPTX exported | PowerPoint download |
| Images exported | Slide images downloaded |

## Email Notifications

Receive notifications via email in addition to in-app alerts.

### Immediate Notifications

For time-sensitive events:
- Direct mentions
- Comment replies
- Ownership transfers
- Sharing invitations

### Configuring Email Notifications

1. Go to **Settings** > **Notifications**
2. Toggle notification types on/off
3. Choose frequency:
   - Immediate
   - Daily summary
   - Weekly digest
   - Off

### Digest Emails

Receive weekly summary emails of activity in your presentations.

See [Digest Emails](/docs/admin/digest-emails/) for full details.

### Digest Content

Weekly digests include:
- Views and engagement stats
- New comments and feedback
- Collaborator activity
- AI-generated insights (if enabled)

### Configuring Digest

1. Go to **Settings** > **Notifications**
2. Toggle **Weekly Digest** on
3. Select preferred day (Monday, Friday, or Sunday)

## Notification Settings

### Per-Presentation Settings

Customize notifications for specific presentations:
1. Open the presentation
2. Go to **Settings** > **Notifications**
3. Choose your preferences:
   - All activity
   - Mentions only
   - None (mute)

### Global Settings

Set your default notification preferences:
1. Go to **Settings** > **Notifications**
2. Configure defaults for new presentations
3. Set email preferences

### Muting Presentations

To stop notifications from a busy presentation:
1. Open the presentation
2. Click **Mute Notifications**
3. You'll still see mentions

## Best Practices

### Stay Organized

- Check notifications regularly
- Clear handled notifications
- Use filters to focus

### Don't Over-Notify

- Use mentions sparingly
- Resolve comments promptly
- Mute presentations you don't need to track

### Configure for Your Workflow

- Set up email notifications to match your habits
- Use digests for less urgent presentations
- Immediate notifications for active collaborations
