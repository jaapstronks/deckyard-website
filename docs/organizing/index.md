# Organizing Presentations

Keep your presentations organized and easy to find.

## In This Section

- **[Search](/docs/organizing/search)** - Find presentations by title, content, or tags
- **[Tags](/docs/organizing/tags)** - Categorize presentations with tags
- **[Trash & Recovery](/docs/organizing/trash)** - Recover deleted presentations

## The Presentation List

The main view (`/app`) shows all your presentations:

![Dashboard with presentation cards and quick start themes](/images/screenshots/dashboard-home-new.png)

- **Thumbnail** - Preview of the first slide
- **Title** - Presentation name
- **Modified date** - Last edit time
- **Language actions** - Quick access to NL/EN versions

## Sorting and Filtering

- Sort by date modified, title, or creation date
- Filter by tag to see only matching presentations
- Search to find specific content
