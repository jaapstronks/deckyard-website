Organize your presentations using tags for easy categorization and discovery.

## What Tags Are For

Tags help you:

- Group related presentations (e.g., "Q1 2025", "Marketing", "Training")
- Find presentations quickly by category
- Create logical collections across different projects

![Deck settings showing Tags field with input for adding tags and Description field for meta descriptions](/images/screenshots/deck-settings-tags-description.png)

## Creating Tags

1. Open a presentation's settings or info panel
2. Click **Add Tag**
3. Type a tag name
4. Press Enter to create

Tags are created on-the-fly as you type them.

## Tagging Presentations

- Add multiple tags to a single presentation
- Remove tags by clicking the X on the tag
- Tags are shared across your workspace

## Filtering by Tag

In the presentation list:

1. Click a tag in the tag filter
2. Only presentations with that tag are shown
3. Click again to remove the filter

## Managing Tags

- Unused tags are automatically cleaned up
- Rename tags by editing them on any presentation
- Tags are case-insensitive ("Marketing" = "marketing")

## Tag Suggestions

Common tag patterns:

- **By project**: "Project Alpha", "Website Redesign"
- **By date**: "Q1 2025", "2025"
- **By type**: "Training", "Sales", "Internal"
- **By status**: "Draft", "Final", "Archive"
