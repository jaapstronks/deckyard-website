# Creating Presentations

Deckyard offers multiple ways to create new presentations, from starting fresh to importing existing content.

![New presentation dialog showing creation options: start blank, from template, or from content with AI](/images/screenshots/new-presentation-dialog.png)

## Methods

- **[New Presentation](/docs/creating/new-presentation)** - Start from scratch with a theme
- **[From Text](/docs/creating/from-text)** - Paste text and let AI generate slides
- **[From Notion](/docs/creating/from-notion)** - Import a Notion page
- **[From PowerPoint](/docs/creating/from-pptx)** - Convert a PPTX file
- **[From Markdown](/docs/creating/from-markdown)** - Import a markdown slide deck (no AI)
- **[From JSON](/docs/creating/from-json)** - Import Deckyard's portable format
- **[Duplicating](/docs/creating/duplicating)** - Copy an existing presentation

## Quick Comparison

| Method | Best For |
|--------|----------|
| New Presentation | Starting fresh with full control |
| From Text | Quickly turning documents into slides |
| From Notion | Teams already using Notion for content |
| From PowerPoint | Migrating existing presentations |
| From Markdown | Developers and version-controlled decks |
| Duplicating | Creating variations or templates |
